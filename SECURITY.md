# Security Policy

## Supported Versions

We support the latest released version of the Go4Garage website.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
**privately** — do not open a public issue.

- Email: **security@go4garage.com** (or hello@go4garage.com if the former is unavailable)
- Please include a clear description, steps to reproduce, and the potential impact.
- We aim to acknowledge reports within **2 business days** and provide a remediation
  plan within **14 days** for confirmed vulnerabilities.

## Scope

In scope:
- This website (go4garage.com and its Firebase Hosting preview URLs)
- Source code in this repository

Out of scope:
- Social engineering, phishing, or physical attacks
- Denial of Service attacks
- Issues already reported or publicly known

Thank you for helping keep Go4Garage and our users safe.
