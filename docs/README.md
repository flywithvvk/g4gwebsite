# Documentation

This directory contains internal reference material and deployment docs for the Go4Garage website.

## Files

| File | Description |
|------|-------------|
| `DEPLOYMENT.md` | How to build and deploy the site (Firebase Hosting, AWS Amplify, or generic Next.js production server) |
| `Go4Garage_Master_Mapping_with_Users_RBAC.xlsx.zip` | Master mapping: products, problems, user personas, RBAC (internal reference, source of truth for content) |
| `extracted_data/` | HTML export of the master mapping spreadsheet for quick browsing without Excel |

## Note

The contents of this directory are **internal reference material** — they inform the website's content (products, problems, personas) but are not themselves published. They are tracked in the repo for authorised collaborators.

If you add new docs here, update this README.
